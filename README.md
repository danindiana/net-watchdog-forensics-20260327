# Net Watchdog Forensics

A suite of tools for NIC hardening, network noise suppression, and automated forensic analysis using local LLMs.

## Project Scope
This project was born out of a network triage session on March 27, 2026, to resolve latency and "poor" connection reports on an Intel X540-T2 NIC.

## Tools
### 1. `harden_nic.sh`
Optimizes the Intel ixgbe driver settings:
- Doubled Ring Buffers (8192)
- Interrupt Coalescing (96us)
- MTU 1500 Enforcement

### 2. `suppress_noise.sh`
Mitigates Layer 2 noise:
- Drops AT&T BGW320 Proprietary Heartbeats (Ethertype 0x7373) using `ebtables`.
- Blocks UPnP/mDNS bloat using `UFW`.

### 3. `analyze_state.py`
The **Ollama-Delegate** implementation:
- Gathers system process and socket state.
- Automatically filters kernel noise.
- Offloads complex forensic pattern matching to local Ollama models (default: `deepseek-r1:14b`).

## Usage
```bash
sudo ./scripts/harden_nic.sh enp3s0f0
sudo ./scripts/suppress_noise.sh
python3 ./scripts/analyze_state.py
```
